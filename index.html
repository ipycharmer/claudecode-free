Features
Everything you need. Nothing you don't.

A complete agentic coding environment that installs once and runs anywhere.

🆓

Completely Free

Uses OpenRouter's free model tier. Owl Alpha has no daily usage limits at all. No credit card required.

🐳

One Build, Run Forever

Docker image builds once in ~10 minutes. Every subsequent launch is instant. Works on macOS 10.15+, Linux, WSL2.

🤖

Full Agentic Power

Reads files, writes code, runs bash, uses git, installs packages, browses the web. Same as Claude Code with Anthropic.

🔀

5 Free Models

Switch between Owl Alpha, Nemotron, Ring, GPT-OSS, and Laguna with arrow keys. Or type any OpenRouter model ID.

🛠️

Full Dev Stack

Python, Node, Go, Rust, sqlite3, Chromium, Playwright, Docker CLI — all pre-installed in the image.

🎨

Beautiful TUI

Animated logo, arrow-key model picker, smooth startup sequence. Looks good in any terminal.

Quick Start
Up in 3 commands.

First run builds the image (~10 min). Every run after that is instant.

1

Clone & make executable

git clone https://github.com/ipycharmer/claudecode-free cd claudecode-free chmod +x run.sh
2

Get a free OpenRouter API key

Go to openrouter.ai/keys and create a free account. No credit card needed for free models.

3

Run it

./run.sh # Enter your API key when prompted (saved for future runs) # Select a model with ↑↓ arrows # Enter your workspace folder path # Claude Code launches — start coding
