// features

Everything you need.

A complete agentic dev environment. Build it once, use it on any project forever.

🆓

Completely Free

Uses OpenRouter's free tier. Owl Alpha has zero daily limits. No credit card, no Anthropic billing.

🐳

One Build, Forever

Docker image builds once in ~10 minutes. Every run after is instant. Works on macOS 10.15+, Linux, WSL2.

🤖

Full Agentic Power

Reads files, writes code, runs bash, uses git, installs packages, browses the web — same as Claude Code with Anthropic.

⌨️

Arrow-Key Model Picker

Navigate 5 curated free models with ↑↓ keys. Press Enter or a number to select. Type any model ID manually.

🛠️

Full Dev Stack

Python, Node, Go, Rust, SQLite, Chromium, Playwright, Docker CLI — pre-installed. The model can use all of it.

🔀

Smart Proxy

Translates Anthropic's Messages API to OpenRouter format. Handles streaming, tool calls, dedup, and identity injection.

// quick start

Up in 3 commands.

First run builds the image (~10–15 min). Every run after is instant.

1

Clone & make executable

git clone https://github.com/ipycharmer/claudecode-free cd claudecode-free chmod +x run.sh
2

Get a free OpenRouter key

Go to openrouter.ai/keys — free account, no credit card needed for free models.

3

Run

./run.sh # enter API key once (saved for future) # pick model with ↑↓ arrows # enter workspace path # Claude Code launches
